# tritask-webapp1
[Tritask](https://tritask.github.io/tritask-web/) をウェブアプリ化してみたものです。

## こちらからアクセス: <https://stakiran.github.io/tritask-webapp1/>

## 注意点
- あくまでプロトタイプなので、個人情報その他重要な情報は扱わないでください
    - データは生で保存しているため、管理者には見えます
- データは Firebase 無料プラン（Spark）を使っており、無料枠を超えたらそれ以上読み書きできません
    - デイリー 1 万回ですが、自動保存やソートで 1 回使うので案外すぐ超えるかも。。。
- （Tritask 利用者向け）本家と比べると色々機能は劣ります、また細かい挙動までは再現しきれていません

## 概要や使い方
- 1行1タスク、テキストベースのタスク管理ツール
- 「今日やった」「今日やる」「明日以降やる」「昨日以前やった」の 4 つのセクションがあります
    - 「今日やった」にタスクを並べて、全部終わらせましょう
    - 明日以降でいいものは「明日以降やる」に移しましょう
    - ソート操作をすると統一的な並びになります
- **最初は意味不明だと思うので、右上サンプルボタンからサンプルデータをロードしてみてください**

元ネタは TaskChute であり、TaskChute をテキストベースで軽快に動かしたいのが動機です。その他詳しい情報は [Tritask](https://tritask.github.io/tritask-web/) を参照。
